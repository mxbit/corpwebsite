Cricket Match Analysis Mobile App

Howzat!

Cricket Match Analysis Software designed to analyze the different aspects in a Live Cricket match such as the Batting, Bowling, Fielding, Partnership, and Over by Over Analysis.

Has a detailed Wagon Wheel with graphics, statistics. Batsmen's strengths and weakness can be analyzed different data visualizations. Created on Android Platform using Adobe Flex for Mobile.

Technology

Actionscript 3.0, FlashBuilder 4.6, SQLite
